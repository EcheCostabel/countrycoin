# Billetera Virtual

## Setup

### 1. Clone repository

```bash
git clone https://github.com/No-Country/c12-31-ft-node-react
```

### 2. Install dependencies

```bash
npm install
```

### 3. Add enviroment variables

```bash
cd server
cp .env.template .env
```

### 4. Run Postgres docker container (optional)

```bash
docker compose up
```

### 5. Run development server

```bash
npm run dev
```
